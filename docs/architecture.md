# Project Architecture

This document describes the overall structure of the Sports Betting Strategy Analyzer project.

## Folder Structure

- `src/` — Core Python modules (data loading, preprocessing, modeling, evaluation)
- `tests/` — Unit and integration tests
- `data/` — Input datasets
- `results/` — Output figures and analysis
- `docs/` — Documentation and design notes

Further details will be added as the project develops.
